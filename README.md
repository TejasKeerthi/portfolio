# Tejas Keerthi — Portfolio

A vibrant AI/ML portfolio built with React 19, Three.js, Framer Motion, and Vite. Features interactive GPU chip schematics, neural network visualizations, data pipeline animations, a paint-brush custom cursor, and smooth Lenis scrolling.

## 🚀 Live Site

**[portfolio-five-neon-54.vercel.app](https://portfolio-five-neon-54.vercel.app)**

## Tech Stack

- **React 19** + **Vite 7**
- **Three.js** / React Three Fiber / Drei
- **Framer Motion** for animations
- **Tailwind CSS v4**
- **Lenis** smooth scrolling
- Deployed on **Vercel**

## Run Locally

```bash
npm install
npm run dev
```
